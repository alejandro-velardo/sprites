This a Sprites game qhere you can create a drawing ainting squares in a canvass.

A color can be selected in the input field and, when the squares of the canvass are clicked, the selected color will be aplied to that square.

When the clean button is clicked, the whole canvass will be painted of the color selected in that moment. 

To run this small app install de dependencies:

npm install

And run the programme by using:

npm run dev
